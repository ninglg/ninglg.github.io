ејәдёҖиҮҙзҡ„еҲҶеёғејҸдәӢеҠЎ

XAеҚҸи®®гҖҒдәҢйҳ¶ж®ө2PCпјҲprepareгҖҒcommitпјүгҖҒдёүйҳ¶ж®ө3PCжҸҗдәӨгҖӮ

XAеҚҸи®®

XA е°ұжҳҜ X/Open DTP е®ҡд№үзҡ„дәӨжҳ“дёӯй—ҙд»¶дёҺж•°жҚ®еә“д№Ӣй—ҙзҡ„жҺҘеҸЈи§„иҢғпјҲеҚіжҺҘеҸЈеҮҪж•°пјүпјҢдәӨжҳ“дёӯй—ҙд»¶з”Ёе®ғжқҘйҖҡзҹҘж•°жҚ®еә“дәӢеҠЎзҡ„ејҖе§ӢгҖҒз»“жқҹд»ҘеҸҠжҸҗдәӨгҖҒеӣһж»ҡзӯүгҖӮ XA жҺҘеҸЈеҮҪж•°з”ұж•°жҚ®еә“еҺӮе•ҶжҸҗдҫӣгҖӮ

X/Open DTP жЁЎеһӢпјҲ 1994 пјүеҢ…жӢ¬еә”з”ЁзЁӢеәҸпјҲ AP пјүгҖҒдәӢеҠЎз®ЎзҗҶеҷЁпјҲ TM пјүгҖҒиө„жәҗз®ЎзҗҶеҷЁпјҲ RM пјүгҖҒйҖҡдҝЎиө„жәҗз®ЎзҗҶеҷЁпјҲ CRM пјүеӣӣйғЁеҲҶгҖӮ

2PC

дәҢйҳ¶ж®өжҸҗдәӨзҡ„з®—жі•жҖқи·ҜеҸҜд»ҘжҰӮжӢ¬дёәпјҡеҸӮдёҺиҖ…е°Ҷж“ҚдҪңжҲҗиҙҘйҖҡзҹҘеҚҸи°ғиҖ…пјҢеҶҚз”ұеҚҸи°ғиҖ…ж №жҚ®жүҖжңүеҸӮдёҺиҖ…зҡ„еҸҚйҰҲжғ…жҠҘеҶіе®ҡеҗ„еҸӮдёҺиҖ…жҳҜеҗҰиҰҒжҸҗдәӨж“ҚдҪңиҝҳжҳҜдёӯжӯўж“ҚдҪңгҖӮз¬¬дёҖйҳ¶ж®өпјҡеҮҶеӨҮйҳ¶ж®ө(жҠ•зҘЁйҳ¶ж®ө)е’Ңз¬¬дәҢйҳ¶ж®өпјҡжҸҗдәӨйҳ¶ж®өпјҲжү§иЎҢйҳ¶ж®өпјүгҖӮ

2PCйқўдёҙзҡ„й—®йўҳпјҡеҗҢжӯҘйҳ»еЎһй—®йўҳпјҢд»ҘеҸҠеҚҸи°ғиҖ…ж•…йҡңеҗҺйҖ жҲҗзҡ„ж•°жҚ®йғЁеҲҶдёҖиҮҙй—®йўҳгҖӮ

3PC

дёүйҳ¶ж®өжҸҗдәӨпјҲThree-phase commitпјүпјҢд№ҹеҸ«дёүйҳ¶ж®өжҸҗдәӨеҚҸи®®пјҲThree-phase commit protocolпјүпјҢжҳҜдәҢйҳ¶ж®өжҸҗдәӨпјҲ2PCпјүзҡ„ж”№иҝӣзүҲжң¬гҖӮ

3PCеҲҶдёәcan_commitгҖҒpre_commitгҖҒcommit/cancelдёүдёӘйҳ¶ж®өгҖӮ

еҰӮжһңеӣ дёәеҚҸи°ғиҖ…жҲ–зҪ‘з»ңй—®йўҳпјҢеҜјиҮҙеҸӮдёҺиҖ…иҝҹиҝҹдёҚиғҪж”¶еҲ°жқҘиҮӘеҚҸи°ғиҖ…зҡ„commitжҲ–rollbackиҜ·жұӮпјҢйӮЈд№ҲеҸӮдёҺиҖ…е°ҶдёҚдјҡеҰӮдёӨйҳ¶ж®өжҸҗдәӨдёӯйӮЈж ·йҷ·е…Ҙйҳ»еЎһпјҢиҖҢжҳҜзӯүеҫ…и¶…ж—¶еҗҺз»§з»ӯcommitгҖӮзӣёеҜ№дәҺдёӨйҳ¶ж®өжҸҗдәӨиҷҪз„¶йҷҚдҪҺдәҶеҗҢжӯҘйҳ»еЎһпјҢдҪҶд»Қз„¶ж— жі•йҒҝе…Қж•°жҚ®зҡ„дёҚдёҖиҮҙжҖ§гҖӮ

жҹ”жҖ§дәӢеҠЎ

ж №жҚ®BASEзҗҶи®әпјҢиҫҫеҲ°жңҖз»ҲдёҖиҮҙжҖ§гҖӮ

DNS е°ұжҳҜдёҖдёӘе…ёеһӢзҡ„жңҖз»ҲдёҖиҮҙжҖ§зі»з»ҹгҖӮ

еңЁе·ҘзЁӢе®һи·өдёҠпјҢдёәдәҶдҝқйҡңзі»з»ҹзҡ„еҸҜз”ЁжҖ§пјҢдә’иҒ”зҪ‘зі»з»ҹеӨ§еӨҡе°ҶејәдёҖиҮҙжҖ§йңҖжұӮиҪ¬жҚўжҲҗжңҖз»ҲдёҖиҮҙжҖ§зҡ„йңҖжұӮпјҢе№¶йҖҡиҝҮзі»з»ҹжү§иЎҢе№ӮзӯүжҖ§зҡ„дҝқиҜҒпјҢдҝқиҜҒж•°жҚ®зҡ„жңҖз»ҲдёҖиҮҙжҖ§гҖӮ

жңҖз»ҲдёҖиҮҙжҖ§

TCCиЎҘеҒҝжңәеҲ¶пјҲTryгҖҒConfirm/Cancelпјү

е…¶ж ёеҝғжҖқжғіжҳҜпјҡй’ҲеҜ№жҜҸдёӘж“ҚдҪңпјҢйғҪиҰҒжіЁеҶҢдёҖдёӘдёҺе…¶еҜ№еә”зҡ„зЎ®и®Өе’ҢиЎҘеҒҝпјҲж’Өй”Җпјүж“ҚдҪңгҖӮ

TCCж–ҪиЎҢжӯҘйӘӨ

  1. Try йҳ¶ж®өдё»иҰҒжҳҜеҜ№дёҡеҠЎзі»з»ҹеҒҡжЈҖжөӢеҸҠиө„жәҗйў„з•ҷгҖӮ
  2. Confirm йҳ¶ж®өдё»иҰҒжҳҜеҜ№дёҡеҠЎзі»з»ҹеҒҡзЎ®и®ӨжҸҗдәӨпјҢTryйҳ¶ж®өжү§иЎҢжҲҗеҠҹе№¶ејҖе§Ӣжү§иЎҢConfirmйҳ¶ж®өж—¶пјҢй»ҳи®Ө Confirmйҳ¶ж®өжҳҜдёҚдјҡеҮәй”ҷзҡ„гҖӮ
  3. Cancel йҳ¶ж®өдё»иҰҒжҳҜеңЁдёҡеҠЎжү§иЎҢй”ҷиҜҜпјҢйңҖиҰҒеӣһж»ҡзҡ„зҠ¶жҖҒдёӢжү§иЎҢзҡ„дёҡеҠЎеҸ–ж¶ҲпјҢйў„з•ҷиө„жәҗйҮҠж”ҫгҖӮ

TCCдёҺ2PCеҚҸи®®жҜ”иҫғпјҡ

  1. дҪҚдәҺдёҡеҠЎжңҚеҠЎеұӮиҖҢйқһиө„жәҗеұӮ
  2. жІЎжңүеҚ•зӢ¬зҡ„еҮҶеӨҮ(Prepare)йҳ¶ж®өпјҢTryж“ҚдҪңе…јеӨҮиө„жәҗж“ҚдҪңдёҺеҮҶеӨҮиғҪеҠӣ
  3. Tryж“ҚдҪңеҸҜд»ҘзҒөжҙ»йҖүжӢ©дёҡеҠЎиө„жәҗзҡ„й”Ғе®ҡзІ’еәҰ(д»ҘдёҡеҠЎе®ҡзІ’еәҰ)
  4. иҫғй«ҳејҖеҸ‘жҲҗжң¬

жң¬ең°ж¶ҲжҒҜиЎЁ

зұ»дјјдәҺеҸҜйқ ж¶ҲжҒҜж–№жЎҲгҖӮ

ж¶ҲжҒҜз”ҹдә§ж–№пјҢйңҖиҰҒйўқеӨ–е»әдёҖдёӘж¶ҲжҒҜиЎЁпјҢе№¶и®°еҪ•ж¶ҲжҒҜеҸ‘йҖҒзҠ¶жҖҒгҖӮж¶ҲжҒҜиЎЁе’ҢдёҡеҠЎж•°жҚ®иҰҒеңЁдёҖдёӘдәӢеҠЎйҮҢжҸҗдәӨпјҢд№ҹе°ұжҳҜиҜҙд»–д»¬иҰҒеңЁдёҖдёӘж•°жҚ®еә“йҮҢйқўгҖӮз„¶еҗҺж¶ҲжҒҜдјҡз»ҸиҝҮMQеҸ‘йҖҒеҲ°ж¶ҲжҒҜзҡ„ж¶Ҳиҙ№ж–№гҖӮеҰӮжһңж¶ҲжҒҜеҸ‘йҖҒеӨұиҙҘпјҢдјҡиҝӣиЎҢйҮҚиҜ•еҸ‘йҖҒгҖӮ

ж¶ҲжҒҜж¶Ҳиҙ№ж–№пјҢйңҖиҰҒеӨ„зҗҶиҝҷдёӘж¶ҲжҒҜпјҢе№¶е®ҢжҲҗиҮӘе·ұзҡ„дёҡеҠЎйҖ»иҫ‘гҖӮжӯӨж—¶еҰӮжһңжң¬ең°дәӢеҠЎеӨ„зҗҶжҲҗеҠҹпјҢиЎЁжҳҺе·Із»ҸеӨ„зҗҶжҲҗеҠҹдәҶпјҢеҰӮжһңеӨ„зҗҶеӨұиҙҘпјҢйӮЈд№Ҳе°ұдјҡйҮҚиҜ•жү§иЎҢгҖӮеҰӮжһңжҳҜдёҡеҠЎдёҠйқўзҡ„еӨұиҙҘпјҢеҸҜд»Ҙз»ҷз”ҹдә§ж–№еҸ‘йҖҒдёҖдёӘдёҡеҠЎиЎҘеҒҝж¶ҲжҒҜпјҢйҖҡзҹҘз”ҹдә§ж–№иҝӣиЎҢеӣһж»ҡзӯүж“ҚдҪңгҖӮ

з”ҹдә§ж–№е’Ңж¶Ҳиҙ№ж–№е®ҡж—¶жү«жҸҸжң¬ең°ж¶ҲжҒҜиЎЁпјҢжҠҠиҝҳжІЎеӨ„зҗҶе®ҢжҲҗзҡ„ж¶ҲжҒҜжҲ–иҖ…еӨұиҙҘзҡ„ж¶ҲжҒҜеҶҚеҸ‘йҖҒдёҖйҒҚгҖӮ

дјҳзӮ№пјҡ дёҖз§Қйқһеёёз»Ҹе…ёзҡ„е®һзҺ°пјҢйҒҝе…ҚдәҶеҲҶеёғејҸдәӢеҠЎпјҢе®һзҺ°дәҶжңҖз»ҲдёҖиҮҙжҖ§гҖӮ

зјәзӮ№пјҡ ж¶ҲжҒҜиЎЁдјҡиҖҰеҗҲеҲ°дёҡеҠЎзі»з»ҹдёӯпјҢеҰӮжһңжІЎжңүе°ҒиЈ…еҘҪзҡ„и§ЈеҶіж–№жЎҲпјҢдјҡжңүеҫҲеӨҡжқӮжҙ»йңҖиҰҒеӨ„зҗҶгҖӮ

дәӢеҠЎж¶ҲжҒҜ

йҖҡиҝҮдҪҝз”ЁRocketMQпјҢз¬¬дёҖйҳ¶ж®өеҸ‘йҖҒPreparedж¶ҲжҒҜж—¶пјҢдјҡжӢҝеҲ°ж¶ҲжҒҜзҡ„ең°еқҖпјҢз¬¬дәҢйҳ¶ж®өжү§иЎҢжң¬ең°дәӢеҠЎпјҢз¬¬дёүйҳ¶ж®өйҖҡиҝҮз¬¬дёҖйҳ¶ж®өжӢҝеҲ°зҡ„ең°еқҖеҺ»и®ҝй—®ж¶ҲжҒҜпјҢе№¶дҝ®ж”№ж¶ҲжҒҜзҡ„зҠ¶жҖҒгҖӮ

е…¶д»–

  1. XAеҚҸи®®жҳҜдёҖдёӘеҹәдәҺж•°жҚ®еә“зҡ„еҲҶеёғејҸдәӢеҠЎеҚҸи®®пјҢе…¶еҲҶдёәдёӨйғЁеҲҶпјҡдәӢеҠЎз®ЎзҗҶеҷЁе’Ңжң¬ең°иө„жәҗз®ЎзҗҶеҷЁгҖӮдәӢеҠЎз®ЎзҗҶеҷЁдҪңдёәдёҖдёӘе…ЁеұҖзҡ„и°ғеәҰиҖ…пјҢиҙҹиҙЈеҜ№еҗ„дёӘжң¬ең°иө„жәҗз®ЎзҗҶеҷЁз»ҹдёҖеҸ·д»ӨжҸҗдәӨжҲ–иҖ…еӣһж»ҡгҖӮдәҢйҳ¶жҸҗдәӨеҚҸи®®пјҲ2PCпјүе’Ңдёүйҳ¶жҸҗдәӨеҚҸи®®пјҲ3PCпјүе°ұжҳҜж №жҚ®жӯӨеҚҸи®®иЎҚз”ҹеҮәжқҘиҖҢжқҘгҖӮзӣ®еүҚ OracleгҖҒMysql зӯүж•°жҚ®еә“еқҮе·Іе®һзҺ°дәҶXAжҺҘеҸЈгҖӮ
  2. дёӨж®өжҸҗдәӨпјҲ2PCпјүе°ұжҳҜиҝӣиЎҢдёӨдёӘйҳ¶ж®өзҡ„жҸҗдәӨпјҡз¬¬дёҖйҳ¶ж®өпјҢеҮҶеӨҮйҳ¶ж®ө(жҠ•зҘЁйҳ¶ж®ө) пјӣз¬¬дәҢйҳ¶ж®өпјҢжҸҗдәӨйҳ¶ж®өпјҲжү§иЎҢйҳ¶ж®өпјүгҖӮ
  3. дёүж®өжҸҗдәӨпјҲ3PCпјүжҳҜеҜ№дёӨж®өжҸҗдәӨпјҲ2PCпјүзҡ„дёҖз§ҚеҚҮзә§дјҳеҢ–пјҢ3PCеңЁ2PCзҡ„з¬¬дёҖйҳ¶ж®өе’Ңз¬¬дәҢйҳ¶ж®өдёӯжҸ’е…ҘдёҖдёӘеҮҶеӨҮйҳ¶ж®өгҖӮдҝқиҜҒдәҶеңЁжңҖеҗҺжҸҗдәӨйҳ¶ж®өд№ӢеүҚпјҢеҗ„еҸӮдёҺиҖ…иҠӮзӮ№зҡ„зҠ¶жҖҒйғҪдёҖиҮҙгҖӮеҗҢж—¶еңЁеҚҸи°ғиҖ…е’ҢеҸӮдёҺиҖ…дёӯйғҪеј•е…Ҙи¶…ж—¶жңәеҲ¶пјҢеҪ“еҸӮдёҺиҖ…еҗ„з§ҚеҺҹеӣ жңӘж”¶еҲ°еҚҸи°ғиҖ…зҡ„commitиҜ·жұӮеҗҺпјҢдјҡеҜ№жң¬ең°дәӢеҠЎиҝӣиЎҢcommitпјҢдёҚдјҡдёҖзӣҙйҳ»еЎһзӯүеҫ…пјҢи§ЈеҶідәҶ2PCзҡ„еҚ•зӮ№ж•…йҡңй—®йўҳпјҢдҪҶ3PC иҝҳжҳҜжІЎиғҪд»Һж №жң¬дёҠи§ЈеҶіж•°жҚ®дёҖиҮҙжҖ§зҡ„й—®йўҳгҖӮ
  4. 3PC зҡ„дёүдёӘйҳ¶ж®өеҲҶеҲ«жҳҜCanCommitгҖҒPreCommitгҖҒDoCommitгҖӮ
  5. TCCдёҺ2PCзҡ„жҖқжғіеҫҲзұ»дјјпјҢдәӢеҠЎеӨ„зҗҶжөҒзЁӢд№ҹеҫҲзұ»дјјпјҢдҪҶ2PC жҳҜеә”з”ЁдәҺеңЁDBеұӮйқўпјҢTCCеҲҷеҸҜд»ҘзҗҶи§ЈдёәеңЁеә”з”ЁеұӮйқўзҡ„2PCпјҢжҳҜйңҖиҰҒејҖеҸ‘иҖ…зј–еҶҷдёҡеҠЎйҖ»иҫ‘жқҘе®һзҺ°зҡ„гҖӮ
  6. TCCзҡ„ж ёеҝғжҖқжғіжҳҜпјҡ"й’ҲеҜ№жҜҸдёӘж“ҚдҪңйғҪиҰҒжіЁеҶҢдёҖдёӘдёҺе…¶еҜ№еә”зҡ„зЎ®и®ӨпјҲTryпјүе’ҢиЎҘеҒҝпјҲCancelпјү"гҖӮ
