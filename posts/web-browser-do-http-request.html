浏览器的一次网络请求流程

2011-03-02

介绍一下在浏览器里完成一次网络请求的流程步骤。

当在浏览器里输入一行网址，并按下回车执行之后，会经过以下几个流程：

1. DNS解析

将域名解析成IP地址。

2. TCP连接

TCP三次握手，建立TCP连接。

3. HTTP请求

发送HTTP请求并获取数据结果。

4. 构建DOM树

使用HTML字符串构建DOM树。

5. 构建CSSOM树

使用CSS字符串构建CSSOM树。

6. 合成并绘制

使用DOM+CSSOM进行渲染，合成并绘制结果。
