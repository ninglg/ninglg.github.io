HBase介绍

2015-10-05
大数据

此篇介绍一下HBase相关的内容。

安装

brew install hbase

启动

brew services start hbase

检查

jps

95425 Main
80529
89587
98501 Jps
98459 HMaster

注：jps（Java Virtual Machine Process Status Tool）是JDK 1.5提供的一个显示当前所有java进程pid的命令，简单实用，非常适合在linux/Unix平台上查看当前Java进程的一些简单情况。

命令行交互

hbase shell
